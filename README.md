# Apple-Iphone

A modern web app showcasing interactive 3D views of iPhone models using Three.js and GSAP. This project allows users to explore different iPhone models with smooth animations and color options.

## Features
- **Interactive 3D Models**: View different iPhone models in 3D with zoom and rotation features.
- **Color Selection**: Users can choose between multiple color options for each model.
- **Smooth Animations**: Transitions between different views and models using GSAP.
- **Responsive Design**: Optimized for both desktop and mobile devices.

## Technologies Used
- **React**: For building the user interface.
- **Three.js**: For rendering 3D models.
- **GSAP**: For creating animations.
- **Tailwind CSS**: For styling.
